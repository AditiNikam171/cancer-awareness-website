Cancer Awareness & Support Web Page-
A simple and responsive web page built to raise awareness about cancer and provide support resources. The website features a clean landing page, a contact form, and real-time motivational quotes fetched from a public API.

 Features-
Landing Page: Includes a short awareness message and a banner image to grab attention.
Contact Form: Simple form with Name, Email, and Message fields (no backend required).
Responsive Design: Works smoothly on desktop, tablet, and mobile screens.
Live Quotes Section (Bonus): Displays real-time motivational quotes.

Technologies Used-
HTML5 – Structure of the web page
CSS3 – Styling and responsive layout
JavaScript (ES6) – Fetching and displaying live quotes
